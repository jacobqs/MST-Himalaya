***********************
Package shyft.dashboard
***********************

The Python package ``shyft.dashboard`` provides `Bokeh <https://bokeh.org/>`_ based generic web dashboard components
to display time series.

It is platform independent and can be installed the following way:

1. using **pip** from `PyPI <https://pypi.org/project/shyft.dashboard/>`_

   .. code-block:: bash

      pip install shyft.dashboard

2. using **anaconda/conda** from the `Shyft channel <https://anaconda.org/shyft-os/shyft.dashboard>`_

   .. code-block:: bash

      conda install -c shyft-os shyft.dashboard

.. note::

   This package requires :ref:`PackageTimeSeries`.
